## 2. 섹션별 상세 구조

### 2-1. Top Notice Bar
- 배경: 흰색 또는 연한 회색
- 텍스트: 화살표(`<`, `>`) + 프로모션 문구 (좌우 네비게이션)
- 위치: 페이지 최상단 고정
- 높이: ~30px

### 2-2. Header
```
┌─────────────────────────────────────────────────────────────┐
│ [Notice] [Q&A] [Review]                  [Login][Register] │
│ [Search] [Cart(0)] [MyPage] [KR/GLOBAL]                    │
├─────────────────────────────────────────────────────────────┤
│                     THENCE                                  │
│              (로고, 페이지 중앙)                               │
├─────────────────────────────────────────────────────────────┤
│  About  Lookbook  Archive  Stockist                        │
└─────────────────────────────────────────────────────────────┘
```
- 3단 구성: 유틸리티 바 / 로고 / 메인 네비
- 폰트: 모노스페이스 또는 고딕 계열 영문

### 2-3. Category Navigation
```
[ New Arrivals ] [ Best ] [ Stationery ] [ Collect ] [ Sticker ] [ Card ] [ Apparel ] [ Bags ] [ Acc ] ...
```
- 수평 스크롤 가능한 카테고리 탭
- 현재 선택된 카테고리 밑줄 또는 굵게 표시

### 2-4. Hero Banner
- 전폭 배너
- 배경: 연한 파란색 계열 또는 흰색 + 그래픽 요소
- 주요 텍스트:
  - `THENCE × A-ZIT`
  - `New Season 10% OFF`
  - `6.26 (FRI) ~ 7.05 (SUN)`
- 우측: 콜라주 스타일 그래픽/일러스트

### 2-5. Product Grid
```
Section Title: "New Arrivals"  [더보기 →]
```

## 3. 디자인 시스템

### 컬러 팔레트
| 역할 | 색상 | 코드 |
|------|------|------|
| 배경 | 흰색 | `#FFFFFF` |
| 주요 텍스트 | 검정 | `#000000` |
| 서브 텍스트 | 회색 | `#666666` |
| 취소선 가격 | 밝은 회색 | `#999999` |
| 할인 배지 | 파란색 계열 | `#4A90E2` |
| Notice Bar 배경 | 연회색 | `#F5F5F5` |
| 배너 포인트 | 밝은 파랑 | `#87CEEB` |

### 타이포그래피
| 역할 | 폰트 | 크기 | 굵기 |
|------|------|------|------|
| 로고 | 모노스페이스/Sans | 24px | Bold |
| H1 (배너) | Sans-serif | 36px | Bold |
| 카테고리 | Sans-serif | 13px | Regular |
| 상품명 | Sans-serif | 12px | Regular |
| 가격 | Sans-serif | 13px | Medium |
| 푸터 | Sans-serif | 11px | Regular |

### 간격 (Spacing)
- 컨테이너 최대 너비: 1200px
- 좌우 패딩: 20px (모바일), 40px (PC)
- 상품 그리드 gap: 16px
- 섹션 간격: 48px

### 반응형 브레이크포인트
| 구분 | 너비 | 그리드 열 |
|------|------|----------|
| Mobile | ~768px | 2열 |
| Tablet | 768-1024px | 3열 |
| Desktop | 1024px+ | 4열 |

## 4. 인터랙션

- **Top Notice Bar**: 좌우 화살표로 공지 슬라이드
- **Hero Banner**: 자동 슬라이드 (3-5초 간격)
- **상품 카드 호버**: 이미지 살짝 확대 또는 두 번째 이미지로 전환
- **카테고리 탭**: 클릭 시 해당 섹션으로 스크롤 또는 필터링
- **헤더**: 스크롤 시 고정(sticky)
- **장바구니**: 상품 카드 호버 시 "담기" 버튼 표시
