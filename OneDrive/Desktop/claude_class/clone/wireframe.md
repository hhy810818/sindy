## 디자인 레퍼런스 (Design Tokens)

`css/style.css`의 `:root` 한 곳만 수정하면 전체 반영.

| 토큰 | 값 | 역할 |
|---|---|---|
| `--color-primary` | `#7FC4E8` | 포인트(예약·강조) 파스텔 블루 |
| `--color-accent` | `#DAECF2` | 옅은 하늘 / 보더 |
| `--color-bg-soft` | `#F6FAFD` | 섹션 교차 배경 |
| `--color-dark` | `#2B2B2B` | 다크 섹션·푸터 |
| `--color-text` | `#333` / `--color-text-muted` `#888` | 본문/보조 |
| `--font-heading` | `'Playfair Display', serif` | 영문 장식 헤드라인 |
| `--font-body` | `'Noto Sans KR', sans-serif` | 한글 본문 |
| `--radius` `16px` / `--maxw` `1140px` | | 카드 라운드 / 콘텐츠 폭 |

### 모션
- 스크롤 진입 `fade-in`(아래→제자리)
- 카드 호버 `translateY(-6px)` + 이미지 `scale(1.06)`
- 히어로 자동 슬라이더 5초

> 이미지는 교육용으로 Airbnb(muscache) 공개 CDN을 참조. 완전 독립 사용 시 `img/` 폴더로 내려받아 교체 권장.
